# Dijkstra-Pathfinding-Visualizer
**Ref:** SSBI-001 // Logic-Labs

A high-performance interactive tool designed to visualize the efficiency of the Dijkstra algorithm. Built with a focus on industrial aesthetics and clean computational logic.

### 🧠 Logic Overview
The system utilizes a weighted-node graph theory approach. It calculates the shortest distance from a start node to a target node by traversing neighbors and updating distances in $O(V^2)$ or $O(E \log V)$ efficiency.

### 🛠️ Technical Stack
* **Language:** Vanilla JavaScript (ES6+)
* **Engine:** Custom DOM-Grid Controller
* **Aesthetic:** Industrial Concrete / Safety Orange (High Contrast)

### ⚡ Operational Guide
1. **Define Obstacles:** Click/Drag on the grid to place black "carbon" walls.
2. **Initialize Solver:** Executes the Dijkstra search protocol.
